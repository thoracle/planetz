Cache Clear Instructions

The debug spam reduction changes are in the code but your browser is using cached JavaScript files.

Quick Fix:

  1. Hard Refresh: Press Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac)
  2. Or Clear Cache: Open DevTools (F12) → Network tab → Check "Disable cache" → Refresh
  3. Or Force Reload: Right-click refresh button → "Empty Cache and Hard Reload"

Verify the Fix:

After clearing cache, you should see:

  • ✅ 🔍 Discovery status changed: A0_sol -> false (only once, not repeated)
  • ✅ 🎯 RETICLE: Undiscovered object "Sol" showing as "Unknown" (with target name, only once)
  • ✅ Much less spam overall
