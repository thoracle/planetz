🎮 Card Inventory Drag & Drop Interface

Interactive card collection and ship configuration system

🎯 How to Use the Interface

  • Ship Type: Select different ship types from the dropdown to see ship configurations
  • Drag Cards: Click and drag cards from the inventory to ship slots
  • Drop Zones: Valid slots glow green, invalid slots glow red
  • Remove Cards: Click the × button on installed cards to remove them
  • Upgrade Cards: Click UPGRADE button when you have enough cards
  • Open Packs: Click "Open Card Pack" to get 5 random cards
  • Filter Cards: Use dropdown filters to find specific cards
  • Save/Load: Save your ship configuration to localStorage
  • Ship Stats: View total power and mass of your configuration
