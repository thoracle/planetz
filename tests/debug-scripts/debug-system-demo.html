🔧 Smart Debug Logging System

Browser Console Commands

Use these commands in your browser's developer console:

// List all available channels debugList() // Toggle channels on/off debugToggle('🎯 TARGETING', false) // Disable targeting logs debugToggle('🧪 TESTING', true) // Enable testing logs // Quick enable/disable debugEnable('⚔️ COMBAT') debugDisable('🔍 INSPECTION') // View statistics debugStats() // Reset to defaults debugReset()

💡 Usage Examples

Basic Usage

// Instead of console.log debug('🎯 TARGETING', 'Target acquired successfully'); debug('🚀 MISSIONS', 'Mission objectives updated'); debug('🔴 P1', 'CRITICAL: System failure detected');

Replacing Existing Console Logs

// Before (creates spam) console.log('🎯 TargetComputerManager initialized'); console.log('🎯 Targets detected while monitoring'); // After (channel-based, controllable) debug('🎯 TARGETING', 'TargetComputerManager initialized'); debug('🎯 TARGETING', 'Targets detected while monitoring');

P1 High Priority Channel
